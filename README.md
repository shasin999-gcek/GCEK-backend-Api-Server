# TestApp Backend

## Software Requirements

   

 1. `php>=7.0`
  

 2.  `apache2`


 3. ` mysql`


 4. `composer` 

## Install the Application

  

  * `git clone https://github.com/shasin999-gcek/testapp.git`
  

  * `cd testapp`
  

  * `composer install`
  
  

> **After that add database credintials to config/development.php**

  
  * `php installapp`


  * `composer start `
